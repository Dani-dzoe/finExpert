# FinExpert (Rule-Based Loan Approval Expert System)

Implements:
- User Interface (React)
- Knowledge Base (IF-THEN rules)
- Inference Engine (forward chaining)
- Explanation Module (reasons + fired rule IDs)
- MongoDB storage of applications + decisions

## Requirements
- Node.js 18+
- MongoDB running locally (or a MongoDB URI)

## Run Backend
- cd backend
- npm install
- npm run dev

Backend: http://localhost:5000/health

## Run Frontend
- cd ../frontend
- npm install
- npm run dev

Frontend: http://localhost:5173
